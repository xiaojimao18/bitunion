#BitUnion客户端

Bitunion是北京理工大学FTP联盟论坛，也是北京理工大学最为活跃的论坛之一。
后来论坛开放了接口，加上移动互联网的日益流行，所以在此基础上开发安卓的客户端。

其实在此之前已经有大神finalion开发过[BITUNION](https://github.com/finalion/bitunion)客户端。
但是使用起来还是有一些问题。所以也算是抱着学习的态度，来开发这个客户端。
本人以前没有开发过android的项目，写得不好，还请见谅。

再次感谢大神finalion的开源，给了很多帮助。
